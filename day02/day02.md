### Ep04

- 数据类型

  > 浮点型（在部分输出情况时会出现精度丢失）
  >
  > > ```
  > > #include<iostream>
  > > int main(){
  > > 	float a = 1.23456789e10;
  > > 	//此处会造成精度丢失 应用double
  > > 	b = a+20;
  > > 	cout<<"a ="<< a << "b = "<<b<<endl；
  > > 	return 0;
  > > }
  > > ```
  >

- 混合运算

  > - 短字节->长字节	不需要强制类型转换
  >
  > - 长字节->短字节	需要强制类型转换
  >
  > - 数值按照int运算
  >
  > - 移位运算符
  >
  > > - 数值在运算时候为四个字节运算
  > > 
  > 	  > ```C++
  >     >   > char a,b,c;          0x93 二进制为 1001 0011 0x93
  > 	b = 0x93;                   左移为 0010 0110 0x26
  > 	  > //c = b <<1>>1;此时c为0x93   右移为 0001 0011 0x13
  > 	  > a = b<<1; //此时a为0x26
  > 	  > c = a>>1; //此时c0x13
  > 	  > ```
  >
  > - 防止运算时数值溢出：在计算之前进行强制类型转换
  > - 在混合多种数据时：等号两边的类型要自己用强制类型转换
