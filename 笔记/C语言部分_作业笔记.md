- 3. 寻找最大值和次大值

     - 取出数组的前两个元素，先比较第一个和第二个的大小
     - 循环取出
       - 如果取出元素比最大值大，则更新最大值，将最大值给次大值
       - 如果取出元素比最大值小比次大值，则更新次大值
       - 如果取出元素比最大值和次大值都小，则都不更新

- 4. 超过一半的数字

     - cnt 初始化为0，flag 初始化为a[0]
     - 循环取出元素 
       - 如果 `a == flag`    cnt ++
       - 否则则将     cnt--
       - 如果cnt<0=>flag =a
     - 一个数组内/超过一半出现次数的数/的出现次数一定大于不出现次数
